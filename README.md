# ❤️ Cardiovascular Health AI System

## 📌 Overview
This project is an AI-based cardiovascular health prediction and recommendation system.

It predicts:
- Health Score
- Risk Probability
- Risk Level
- Final Decision
- Recovery & Stress

It also provides:
- Personalized roadmap (1, 3, 6 months)

---

## 🧠 Models Used
- Random Forest Regressor → Health Score
- Logistic Regression → Risk Prediction

---

## ⚙️ Tech Stack
- Python
- FastAPI
- Scikit-learn
- HTML / Tailwind CSS
- JavaScript
